# AquaPredict – AI Based Water Consumption & Shortage Prediction System

Author: Lekkala Leela

## Description
AquaPredict is a web application that analyzes water consumption data and predicts future water demand using machine learning. It helps identify possible water shortages and supports better water management.

## Technologies Used
Python  
Flask  
SQLite  
Pandas  
NumPy  
Scikit-learn  
HTML, CSS, JavaScript  
Chart.js  

## Features
- Add daily water usage data
- Store data in database
- Visualize water usage trends
- Generate charts
- Predict future water consumption
