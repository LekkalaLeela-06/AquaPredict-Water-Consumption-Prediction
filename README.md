# AquaPredict – AI Based Water Consumption Prediction System

This project analyzes water consumption data and predicts future usage using machine learning.

## Technologies Used
Python  
Flask  
Pandas  
NumPy  
Scikit-learn  
Chart.js  

## Run Project

python -m venv venv  
source venv/bin/activate  
pip install -r requirements.txt  
python app.py

Open:
http://127.0.0.1:5000
